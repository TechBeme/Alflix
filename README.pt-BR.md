## 🎯 Visão Geral

**ALLFLIX** é um projeto de portfólio profissional full-stack que demonstra a implementação de uma interface moderna de plataforma de streaming. Construído com tecnologias de ponta, apresenta desenvolvimento frontend avançado, arquitetura backend, design de banco de dados e integrações de API.

**Principais capacidades:**
- Interface moderna e responsiva inspirada nas principais plataformas de streaming
- Sistema completo de autenticação com gerenciamento seguro de sessão
- Integração com API TMDB para 500.000+ filmes e séries
- Funcionalidade avançada de busca e filtragem
- Gerenciamento de perfil de usuário e recursos de lista de favoritos
- Otimizado para performance com Next.js 16 App Router
- Estrutura de código pronta para produção com melhores práticas

### Interface do Usuário e Experiência
- **Landing Page Moderna**: Seção hero com animações suaves e CTAs convincentes
- **Navegação Responsiva**: Navbar mobile-first com busca, notificações e menu de perfil

## 🚀 Início Rápido

### Pré-requisitos

Antes de começar, certifique-se de ter:
- **Node.js 20+** instalado ([Download](https://nodejs.org/))
- **Banco de dados PostgreSQL** (local ou provedor na nuvem como Neon, Supabase, Railway)
- **Chave da API TMDB** (gratuita em [https://www.themoviedb.org/settings/api](https://www.themoviedb.org/settings/api))

### Opção 1: Desenvolvimento Local (Recomendado)

```bash
# 1. Clone o repositório
git clone https://github.com/TechBeme/allflix.git
cd allflix

# 2. Instale as dependências
npm install

# 3. Configure as variáveis de ambiente
cp .env.example .env.local
# Edite .env.local com suas credenciais (veja seção Configuração)

# 4. Configure o banco de dados
npm run db:generate    # Gera o cliente Prisma
npm run db:push        # Envia o schema para o banco de dados
npm run db:seed        # (Opcional) Popula com dados de exemplo

# 5. Execute o servidor de desenvolvimento
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) para ver a aplicação.

### Opção 2: Build de Produção

```bash
# Build para produção
npm run build

# Inicia servidor de produção
npm run start
```

### Opção 3: Gerenciamento de Banco de Dados

```bash
# Abre o Prisma Studio (GUI do Banco de Dados)
npm run db:studio

# Cria uma nova migration
npm run db:migrate

# Veja o banco de dados no navegador em http://localhost:5555
```

## ⚙️ Configuração

### Variáveis de Ambiente

Crie um arquivo `.env.local` no diretório raiz:

```bash
# Banco de Dados (PostgreSQL)
# Exemplo para Neon: postgresql://user:password@ep-xxx.us-east-2.aws.neon.tech/allflix?sslmode=require
# Exemplo para local: postgresql://postgres:password@localhost:5432/allflix
DATABASE_URL="postgresql://user:password@host:5432/allflix"

# API TMDB (obtenha chave gratuita em https://www.themoviedb.org/settings/api)
TMDB_API_KEY="sua_chave_api_tmdb_aqui"
TMDB_API_BASE_URL="https://api.themoviedb.org/3"

# Autenticação (gere strings aleatórias seguras)
BETTER_AUTH_SECRET="sua-chave-secreta-minimo-32-caracteres"
BETTER_AUTH_URL="http://localhost:3000"

# Next.js
NEXT_PUBLIC_APP_URL="http://localhost:3000"
NODE_ENV="development"
```

### Opções de Banco de Dados

O projeto suporta múltiplos provedores PostgreSQL:

| Provedor | Tipo | Melhor Para |
|----------|------|-------------|
| **[Neon](https://neon.tech/)** | Serverless | Auto-scaling, tier gratuito generoso |
| **[Supabase](https://supabase.com/)** | Gerenciado | PostgreSQL + extras (auth, storage) |
| **[Railway](https://railway.app/)** | Gerenciado | Deploy simples, preços justos |
| **[Vercel Postgres](https://vercel.com/storage/postgres)** | Serverless | Integrado com hospedagem Vercel |
| **PostgreSQL Local** | Auto-hospedado | Controle total, custos zero |

### Comandos Disponíveis

```bash
# Desenvolvimento
npm run dev              # Inicia servidor de desenvolvimento (http://localhost:3000)
npm run build            # Build para produção
npm run start            # Inicia servidor de produção
npm run lint             # Executa ESLint
npm run lint:fix         # Corrige erros de linting automaticamente
npm run type-check       # Executa verificação de tipos TypeScript

# Banco de Dados
npm run db:generate      # Gera o Prisma Client
npm run db:push          # Envia alterações do schema para o banco de dados
npm run db:migrate       # Cria e aplica migrations
npm run db:seed          # Popula banco de dados com dados de exemplo
npm run db:studio        # Abre Prisma Studio (GUI do banco de dados)

# Testes
npm run test             # Executa todos os testes
npm run test:watch       # Executa testes em modo watch

# Análise
npm run analyze          # Analisa tamanho do bundle
```
