## 🚀 Quick Start

1. Clone the repository
2. Copy `.env.example` to `.env.local`
3. Fill in your environment variables
4. Run `npm install`
5. Run `npx prisma db push`
6. Run `npx tsx prisma/seed.ts`
7. Run `npm run dev`
